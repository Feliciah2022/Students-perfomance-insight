# Students-perfomance-insight

This project analyzes student performance data using Python, data visualization tools, and Looker Studio.
The goal is to uncover insights related to students‚Äô math, reading, and writing scores while exploring how gender, ethnicity, parental education, lunch type, and test preparation affect overall performance.

 Project Overview
The Students Performance Dashboard provides a visual and interactive way to understand student outcomes.

It includes:
Overall dataset statistics

Performance comparisons by demographic groups
Subject-wise distribution
Pass/Fail segmentation
Impact of test preparation courses
Average scores by category
The dashboard was built to simplify large datasets into clear insights for educators, researchers, and decision-makers.
